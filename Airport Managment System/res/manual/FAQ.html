Frequently Asked Questions

Below is a list of problems users might frequently ask about the program.

Configuration

I accidentally entered the airport details incorrectly, can I correct this without starting from scratch?

Yes, at any time you can edit the airport configuration by selecting Edit -> Airport.

The file I'm trying to import is showing an error "Invalid XML file". How can I fix this?

Make sure than you are selecting the correct file to load, this must be in XML format and be the right kind of data.
For example, loading an  airport from an obstacle XML file will not work.
If the file you are selecting is the right one, your file may have been corrupted. Re-enter the values manually and export the file again to fix this.

Calculations

I do not understand the calculations being done, any way I can see a break down of what is happening?

Yes, to view a break down of what is happening you must select the calculations tab at the top right-hand side of the screen.
You will then be shown a textual breakdown of the calculations being performed.

Can I export the calculation breakdown for later use?

Yes, calculations can be exported to text file format. To do this, simply bring up the calculations you want to export and select File -> Export -> Calculations.
You can then select the text file you wish to export to.

My airport uses different blast distances or slope ratios, does this application support changing these?

This is supported, these values can be changed in the bottom panel of the left-hand section of the screen. Simply input the values you desire and click the "update" button next to them.

Visualisations

I have dragged the view far away from the runway, how do I recentre it?

To recenter the runway, simple click the update button in the left-hand panel. This will reset the views on the right-hand side of the screen.

I am stuck with one of the views maximised, how to I get back to the two views?

To get back to two views, click anywhere within the view. Several icons will appear, you must click the right-most icon.
