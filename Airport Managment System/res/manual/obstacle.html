Obstacle

Once your runways are set up, obstacles may be added to them.
Each runway may contain up to one obstacle.

Creating Obstacles

In order to introduce an obstacle onto the model of our runway, we must load an obstacle onto the runway.

An obstacle can either be defined from scratch or loaded from a set of predefined obstacles. To do the latter, please see the below section "Importing Obstacles".

  1. Select the runway you wish to add the obstacle to.
  2. If an obstacle currently exists, remove the current obstacle from the runway. (See section below "Removing Obstacles").
  3. Select File -> New -> Obstacle.
  4. Enter the obstacle height, distance from centerline and distance from designators.
  5. Click the "Add" button.
The new obstacle should be shown within the interface, the parameters of this obstacle can be viewed in the "Obstacle Information" panel on the left hand side of the screen.

Removing Obstacles

In order to remove obstacles from a runway, the runway must have an obstacle on.

  1. Select the runway you wish to remove the obstacle from.
  2. Click the "Remove" button in the "Obstacle Information" panel on the left hand side of the screen.

Editing Obstacles

Once an obstacle has been loaded, its parameters can be viewed and changed on the left hand side of the screen within the "Obstacle Information" panel.

  1. Select the desired parameter values and change them by deleting them and putting in the amended values.
  2. Click upon the "update" button in the "Obstacle Information" panel.

Importing Obstacles

To make the task easier for frequent users, the obstacle may be selected from a set of pre-defined common objects.

  1. Select File->Open->Obstacle.
  2. Choose the obstacle you desire from the list of XML files.
  3. Click "open".
The obstacle will now be on the runway and can be eddited as normal.

Saving Obstacles

Because it is likely that new types of obstacle could frequently appear, it is useful to be able to export commonly used obstacles to file.
To do this, first have the desired obstacle configuration upon the runway.

  1. Select File->Save->Obstacle.
  2. Choose the file location you wish to export the obstacle to.
  3. Click "save".

This obstacle can now be reimported at any time by importing it. To do this, see the above section "Importing Obstacles".
