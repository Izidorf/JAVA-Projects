Runway

Each airport configuration consists of one or more runways. In order to edit an airports runways, see the airport section.

Each runway has several assosiated properties:

    1. Two runway designators for each direction. You need only enter one of these.
    2. The default runway properties: TORA, TODA, ASDA, LDA and displacement threshold.
    3. Additional properties used for calculations: RESA, strip end.
    4. The perpendicular lengths: Runway width, cleared distance and graded distance.

    • Adding Runway

      In order to add Runway you must be in the "edit airport" or "create airport" windows.

      Below the airport name and code section is the runway list for this airport. To add the airport click "Add New".

    1. Set the runway designator by selecting the number and the side. (e.g. 09L, 29C, 04)
    2. Set the runway properties for each side of the runway.
    3. Set the strip width, cleared distance and graded distance.
    4. Click on the "Add" button.

Removing Runway

In order to add Runway you must be in the "edit airport" or "create airport" windows.

Below the airport name and code section is the runway list for this airport. You will see a list of the runways currently in the airport.

  1. Select the runway you wish to remove on the list.
  2. Click the "Remove" button on the left hand side of the interface.

Editing Runway

In order to add Runway you must be in the "edit airport" or "create airport" windows.

Below the airport name and code section is the runway list for this airport. You will see a list of the runways currently in the airport.

  1. Select the runway you wish to edit on the list.
  2. Click the "Edit" button on the left hand side of the interface.
  3. Correct the runway parameters which you intended to change.
  4. Click the "Add" button.
